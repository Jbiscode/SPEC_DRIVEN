# Part 3: AI 협업의 뼈대 — 4가지 축

| 축 | 역할 | 비유 |
|----|------|------|
| **Spec** | 요구를 정리 | "무엇을 만들지" 정의 |
| **Context** | AI에게 넣을 문맥 설계 | "어떤 정보를 줄지" |
| **Rules** | AI가 항상 따를 원칙 | "어떻게 행동할지" |
| **Skills** | 반복 작업을 재사용 가능한 단위로 | "자주 쓰는 작업 표준화" |

- **Spec**으로 요구를 정리하고  
- **Context**를 넣고  
- **Rules**로 행동을 통제하고  
- **Skills**로 반복 업무를 재사용한다.

이 중에서 Spec·Context는 이미 들어본 이름일 수 있고, Rules·Skills를 조금 더 구체적으로 보자. **먼저 Spec과 Context 개념부터 정리하자.**
